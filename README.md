### Production AI Systems I've Shipped

| System | What It Does | AI Depth |
|--------|-------------|----------|
| **AI ATS Scoring Engine** | Resume PDF → GPT-4 scoring → ranked shortlist with approve/reject | 670-line prompt, 5 job type categories, dynamic weighted criteria |
| **AI Interview Pipeline** | Auto-generates skill-specific MCQ/T-F questions → AI interview summaries → final assessment with success probability | Structured JSON output, retry with exponential backoff |
| **Mood Inference System** | Analyzes listening history tags → infers emotional state → personalized recommendations + motivational quotes | SQS → Lambda → GPT-4 async pipeline, DLQ for failure recovery |
| **Spatial AI Suggestions** | Room dimensions + floor area analysis → AI furniture recommendations matched against inventory | Redis-cached, payload-hashed, dimension-compatible filtering |
| **AI JD Generator** | Structured job data → GPT generates semantic HTML job postings + extracts skills from uploaded PDFs | Custom prompts for parsing, formatting, and skill categorization |
